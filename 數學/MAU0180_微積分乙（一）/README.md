## 讀書方法  
題目多算，考古題刷3年以上，段考基本都能80分以上。
  
然後段考時題目要看仔細，因為八成的錯誤不是源自於你不會算。  
而是你會算但計算錯誤，以及看錯題目條件。
  
最後，不要僥倖**只會考古題內容**，你會翻車的相信我。
